# ZeichnenApp

ZeichnenApp ist eine einfache Webanwendung, mit der Nutzer direkt im Browser zeichnen können. Das Projekt besteht aus einer HTML-Datei, einer JavaScript-Datei und einer CSS-Datei und benötigt keine zusätzliche Installation.

## Features
- Zeichenfläche zum freien Zeichnen mit der Maus
- Auswahl von Farben und Pinselgrößen
- Möglichkeit, das Bild zu speichern oder zu löschen
- Intuitive Benutzeroberfläche

## Projektstruktur
```
index.html     # Hauptseite der Anwendung
script.js      # Logik für das Zeichnen und die Interaktion
style.css      # Gestaltung der Benutzeroberfläche
LICENSE        # Lizenzinformationen
```

## Installation & Nutzung
1. Repository herunterladen oder klonen:
   ```
   git clone <REPO-URL>
   ```
2. Öffne die `index.html` im Browser deiner Wahl.
3. Beginne mit dem Zeichnen!
